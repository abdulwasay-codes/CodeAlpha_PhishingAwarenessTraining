# CodeAlpha_PhishingAwarenessTraining

## Overview
This project was completed as part of the CodeAlpha Cyber Security Internship.

The presentation focuses on phishing attacks, social engineering techniques, phishing email identification, fake websites, and cybersecurity best practices.

## Topics Covered
- Introduction to Phishing
- Types of Phishing Attacks
- How Phishing Works
- Identifying Phishing Emails
- Fake Websites and URL Spoofing
- Social Engineering Techniques
- Real-World Examples
- Prevention and Best Practices
- Interactive Quiz
- Key Takeaways

## Tools Used
- Microsoft PowerPoint / Gamma AI
- Cybersecurity Research
